## Youtube trend app.
[Youtube trend](https://naughty-liskov-921c26.netlify.com/)
顯示各國 Youtube 熱門影片的列表，主要為 React、Redux 串接 API 的練習，以及 middleware 的觀念實作。

## 功能
- 用戶可以選擇地區，顯示該區域的熱門 Youtube 影片列表

## 使用工具
- React、Redux
- Youtube DATA API
- Semantic-UI-React
